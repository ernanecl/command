### Users

add `user`

```BASH
su -
adduser <new_username>
```

&nbsp;
&nbsp;

change user `password`

```BASH
passwd <username>
sudo passwd
```

&nbsp;
&nbsp;

create `root password`

```BASH
sudo passwd root
```
