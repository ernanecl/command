# notepadqq

add notepadqq repository

```BASH
sudo add-apt-repository ppa:notepadqq-team/notepadqq
```

system update

```BASH
sudo apt-get update
```

`notepadqq` install

```BASH
sudo apt-get install notepadqq
```
